# Atividades Práticas - Curso de Programador Web

Este repositório contém as atividades desenvolvidas durante o curso de Programador Web, focando na implementação de interfaces responsivas utilizando frameworks de mercado.

## 🛠️ Tecnologias Utilizadas
* **Bootstrap 5.2**: Utilizado para estruturação inicial e tipografia display.
* [cite_start]**Materialize CSS 1.0.0**: Framework baseado em Material Design para criação de currículo profissional[cite: 1, 3, 17].
* **HTML5**: Estruturação semântica.

## 🚀 Atividades Realizadas

### Atividade 1: Introdução ao Bootstrap
* Configuração do framework via CDN.
* Uso da classe `.container` para organização de layout.
* Implementação de tipografia com classes `display`.

### Atividade 2: Estruturação com Materialize
* [cite_start]Migração do currículo para a estética Material Design[cite: 10].
* [cite_start]Organização do conteúdo utilizando as classes `.section` e `.divider` para separação visual[cite: 121, 123].
* [cite_start]Aplicação do sistema de containers para centralização automática em 70% da largura[cite: 24, 26].

### Atividade 3: Mídia e Alinhamento (Aprimoramento)
* **Imagens:** Uso da classe `.circle` e `.responsive-img` para fotos de perfil arredondadas.
* **Vídeos:** Implementação de vídeo responsivo utilizando a classe `.video-container`.
* **Alinhamento:** Centralização horizontal de títulos e subtítulos com a classe `.center-align`.
* **Conteúdo Técnico:** Inclusão de seção sobre domínio em *Python para Machine Learning*.
